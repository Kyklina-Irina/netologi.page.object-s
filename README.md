# netologi.page.object-s

Автотесты для функции перевода между картами.

[![Build status](https://ci.appveyor.com/api/projects/status/github/Kyklina-Irina/netologi.page.object-s?svg=true)](https://ci.appveyor.com/project/Kyklina-Irina/netologi.page.object-s)

## Описание

Проект реализует автотесты с использованием паттерна Page Object для проверки функции перевода денег между картами.

## Технологии

- Java
- Selenide
- JUnit 5
- Maven
- Appveyor CI

## Как запустить

1. Убедитесь, что запущено приложение `app-ibank-build-for-testers.jar`.
2. Выполните `mvn test`.
